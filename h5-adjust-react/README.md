# H5 改版演示（React + Tailwind CDN）

## 线上预览

- **GitHub Pages**：<https://dan951781659.github.io/visbody-demo/h5-adjust-react/>

## 本地运行

```bash
cd h5-adjust-react
python3 -m http.server 8765
```

浏览器打开：<http://127.0.0.1:8765/>

## 开发调试

在地址后加 `?debug=1` 可显示「正常数据 / 空状态 / 无模型」等场景切换，用于验收空态与骨架屏；正式分享链接不要带该参数。
