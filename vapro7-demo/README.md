# VAPro7 设备端 Demo

静态 HTML Demo，用于评审 VAPro7 设备端测量流程（综合测量、专业体态、单项、测量设置等）。

## 在线访问

- **评审入口**：<https://dan951781659.github.io/visbody-demo/vapro7-demo/>
- **首页**：<https://dan951781659.github.io/visbody-demo/vapro7-demo/home.html>
- **测量设置**：<https://dan951781659.github.io/visbody-demo/vapro7-demo/settings-modes.html>

## 本地预览

在本目录启动静态服务后打开 `index.html`，例如：

```bash
python3 -m http.server 8765
```
