设备端测量流程

本组页面用于评审 VAPro7 设备端用户流程：首页选项目、两阶段测量准备、点击后继续、测量完成后扫码进入 H5 详细报告。

推荐查看顺序
  1. 首页：查看 3 主卡和快捷单项入口。
  2. 完整测量：两阶段准备 → 倒计时 → 一圈测量 → 测量完成页。
  3. 体态精测：自然站立、测量、结果页。
  4. 单项测量：肩部、颈部、平衡独立流程。
  5. 运营设置：调整首页快捷单项、入口显隐与自动跳转。
