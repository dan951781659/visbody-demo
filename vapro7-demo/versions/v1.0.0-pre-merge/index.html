设备端测量流程

本组页面用于评审 VAPro7 设备端用户流程：首页选项目、实时引导、自动推进、测量完成后扫码进入 H5 详细报告。

推荐查看顺序
  1. 首页：查看 3 主卡和快捷单项入口。
  2. 完整测量：上台、体重、扶手、一圈测量、结果页。
  3. 体态精测：自然站立、测量、结果页。
  4. 单项测量：肩部、颈部、平衡独立流程。
  5. 运营设置：调整首页快捷单项和入口显隐。
  6. 结果方案二：现有对比方案，保留摄像头舞台与双选项设计。
  7. 结果方案三：新增摄像头主导式预览页，用于和正式方案一对比。
