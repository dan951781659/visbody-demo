设备端测量流程

本组页面用于评审 VAPro7 设备端用户流程：首页选项目、体成分四宫格或扶手准备、点击后继续、测量完成后扫码进入 H5 详细报告。

推荐查看顺序
  1. 首页：综合测量、专业测量 · 体态测量、单项测量；底部运营图位可替换 demo/assets/home-promo.png。
  2. 综合测量：体成分四宫格（可关）→ 扶手准备 → 倒计时 → 一圈测量 → 测量完成页。
  3. 专业测量：体态测量准备 → 倒计时 → 测量 → 结果。
  4. 单项测量：肩部可走完整流程；颈、平衡为占位卡（或快捷入口进占位页）。
  5. 测量设置：语音播报、体成分准备引导、顺序池与首页快捷单项。
