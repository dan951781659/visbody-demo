设备端测量流程

本组页面用于评审 VAPro7 设备端真实用户流程：首页选项目、测量准备页、姿态识别、倒计时、测量与报告。

推荐查看顺序
  1. 首页：综合测量、体态测量、经典分体模式、专项测量。
  2. 综合测量：测量准备页（可关）→ 站姿识别 → 扶手接触 → 手臂 45° → 倒计时 → 测量 → 完成页。
  3. 专业测量：体态测量准备 → 倒计时 → 测量 → 结果。
  4. 测量设置：语音播报、测量准备页开关、入口显隐与测量顺序。
打开首页 打开测量设置 重置状态
