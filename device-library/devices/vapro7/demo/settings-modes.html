测量设置

门店测量方案

快速应用方案
先选预设再微调：标准综合 / 经典分体 / 综合+体态 / 自定义。

首页测量入口

显示综合测量
首页显示综合测量主入口。
显示体态测量（专业测量）
首页显示体态测量卡片，保留「专业测量」标签。
显示体成分 + 体围
经典分体模式主入口，一次转台采集。
显示体围测量
经典分体模式独立体围入口（Demo 占位）。
显示平衡测量
平衡已归入主测量模式，不再属于专项列表。

专项测量（肩/颈）

控制单项列表、测量顺序池与首页快捷入口可选项。

显示肩部测量
在单项列表与顺序池中可用。
显示颈部测量
占位演示，在单项列表中可用。

至少开启一项时，首页才显示「单项测量」入口。

测量项目顺序

推荐循环顺序
拖拽调整顺序；可从下方添加主测量与专项。已关闭的项目不会进入推荐下一项。
    当前循环顺序
    综合测量 → 专业测量 → 肩部

    首页快捷入口

    快捷单项
    在首页额外显示一个专项快捷卡；仅可选已开启的肩部/颈部。

    综合测量准备

    显示测量准备页
    开启：测量准备页 → 站姿识别 → 扶手接触 → 手臂 45° → 倒计时 → 测量。关闭：跳过测量准备页，从站姿识别开始。

    播报与报告

    外链报告地址
    留空则使用 Demo 内报告详情页。
    语音播报
    关闭后 Demo 全站不播放语音。
