# Moti Physio 竞品借鉴要点

> 基于《Moti Physio-竞品分析 (2).md》提炼，**重点从 3D 模型（肌肉、骨骼）与报告内容** 两方面归纳我方可借鉴内容。

---

## 一、3D 模型（骨骼、肌肉）借鉴要点

### 1.1 骨骼 3D 模型

| 借鉴点 | Moti Physio 做法 | 我方可借鉴 |
|--------|------------------|------------|
| **骨骼模式** | 通过 3D 骨骼模型做姿势失衡的视觉分析；红色高亮表示失衡程度较大的区域 | 建立可旋转、多视角的 3D 骨骼视图，用颜色/高亮区分失衡严重程度 |
| **与列表联动** | 左侧失衡列表按严重度排序，点击某一项后 3D 视图自动定位到对应身体区域 | 报告页「失衡项 ↔ 3D 区域」一一联动，点击列表项即定位到骨骼对应部位 |
| **三状态对比** | 同一套骨骼支持「标准 / 当前 / 恶化」三种状态切换，底部按钮切换，直观展示不干预会怎样 | 提供理想态、当前态、恶化态三种骨骼姿态对比，强化矫正动机 |

### 1.2 肌肉 3D 模型

| 借鉴点 | Moti Physio 做法 | 我方可借鉴 |
|--------|------------------|------------|
| **肌肉色彩编码** | 肌肉模式：红色 = 紧张，蓝色 = 无力；颜色越深程度越重 | 统一采用「红 = 紧张/过度激活」「蓝 = 无力/抑制」的编码规则，并做深浅分级 |
| **理论依据** | 基于 Janda（姿势肌/相位肌）、Anatomy Trains（筋膜经线），点击肌肉可查看相关失衡说明 | 为肌肉状态标注提供可解释依据（如 Janda 综合征、筋膜经线），支持「点击肌肉 → 弹出说明」 |
| **骨骼/肌肉双模式** | 同一 3D 场景下可切换「骷髅模式」「肌肉模式」，两种模式均支持标准/当前/恶化对比 | 同一报告页支持骨骼视图与肌肉视图切换，便于从结构到软组织一层层解读 |

### 1.3 数据与交互

| 借鉴点 | Moti Physio 做法 | 我方可借鉴 |
|--------|------------------|------------|
| **3D 重建来源** | RGB-D 深度数据驱动骨骼 3D 重建，再推肌肉张力/状态 | 明确骨骼 3D 来自深度/视觉数据，肌肉状态来自算法推断并可在报告中标注不确定性 |
| **多视角** | 支持正/侧/背等多视角，脊柱分析单独提供 VCM、Cobb 角等视图 | 3D 模型支持多视角与剖视（如脊柱力线、关键角度的 2D 辅助视图） |

## 二、报告内容借鉴要点

### 2.1 报告结构与入口

| 借鉴点 | Moti Physio 做法 | 我方可借鉴 |
|--------|------------------|------------|
| **计分屏作为入口** | 整体失衡分数 0–100、Top 3 失衡区域及角度、风险等级（轻/中/重），作为报告第一屏 | 报告首页突出「总分 + Top 3 问题 + 风险等级」，用户先看到结论再进详情 |
| **层级递进** | 第一层：分数 + Top 3；第二层：体态视图（照片/Moiré/面部分析/Adam）；第三层：3D 模拟、脊柱分析、前后对比 | 报告按「摘要 → 体态视图 → 3D/脊柱/对比」分层，避免一屏信息过载 |
| **Reference Guide** | 计分屏左上角提供参考角度、参考线、脊柱表示说明，点击 ⓘ/❔ 可查看各指标释义 | 为关键指标提供「参考范围 + 释义」入口，降低专业术语门槛 |

### 2.2 体态与可视化视图

| 借鉴点 | Moti Physio 做法 | 我方可借鉴 |
|--------|------------------|------------|
| **多视图切换** | 面部分析、Moiré（身体轮廓色彩）、纯照片、Adam 试验结果，同一报告内可切换 | 报告内支持多种体态展示方式（照片/轮廓/专项试验），满足不同解读场景 |
| **Moiré 与角度** | 摩尔纹可切换「照片」「角度」两种展示，角度模式下显示用户指标对应角度 | 轮廓/等高类视图与角度数值可切换显示，兼顾直观与量化 |
| **Top 3 与高亮联动** | 点击 Top 3 某一项，用户图片上该项目对应位置高亮并可有动态指示 | 点击高风险项时，在照片/3D 上高亮对应解剖位置，必要时加简单动效 |

### 2.3 脊柱与专项分析

| 借鉴点 | Moti Physio 做法 | 我方可借鉴 |
|--------|------------------|------------|
| **VCM + Cobb 双方法** | VCM 连接脊柱节段中心点看整体力线；Cobb 角用于脊柱侧弯测量；通过下拉菜单切换 | 脊柱报告提供至少一种「整体力线」视图和一种「角度/侧弯」视图，并可切换 |
| **脊柱分析入口** | 从失衡列表选择「综合分析」→「脊柱详细分析」进入 Cobb 角等视图 | 在失衡或体态列表中提供「脊柱专项分析」入口，不淹没在通用报告中 |
| **脊柱裸机模式配置** | 严重脊柱侧弯时可开启裸机模式（设置 > 功能 > 裸机模式）；着装要求：无上衣/胸罩（女性）+ 紧身下装，以减少衣物遮挡、提升脊柱/躯干轮廓识别精度 | 在设置中提供「脊柱裸机模式」开关，并明确适用场景（如疑似或已知中重度侧弯）与测量前着装指引，便于机构在需要时获得更准的脊柱分析结果 |

### 2.4 前后对比与改善呈现

| 借鉴点 | Moti Physio 做法 | 我方可借鉴 |
|--------|------------------|------------|
| **按改善程度排序** | 前后对比时，改善项按「改善幅度从大到小」排序，角度值用蓝色高亮 | 对比报告突出「改善最多的项目」并排序，用颜色区分改善/恶化 |
| **多维度对比** | 支持按全部/正面/侧面/背面/Moiré 等维度切换对比内容 | 前后对比支持按视角或按维度（如仅脊柱、仅下肢）筛选 |
| **3D 骨架动画** | 前后对比中 3D 自动定位到改善区域，底部 Before/After 切换 | 对比页 3D 能聚焦到变化明显的区域，并支持前后状态切换 |

### 2.5 报告导出与分享

| 借鉴点 | Moti Physio 做法 | 我方可借鉴 |
|--------|------------------|------------|
| **多种报告模板** | 骨骼报告表、专家报告表、原始照片报告表、风险排名报告表等，按场景选择 | 提供多套报告模板（如客户版/专业版/科研版），满足不同角色与场景 |
| **分享与隐私** | 分享图支持人脸马赛克、显示机构名称/Logo；左侧选中的视图决定分享内容 | 导出/分享时支持脱敏（如人脸模糊）、品牌露出，且「所见即所分享」 |
| **导出渠道** | 打印、保存、邮件、扫码获取等多种方式 | 支持打印、PDF、邮件、二维码/链接等多种导出与分享方式 |

### 2.6 动态评估与报告

| 借鉴点 | Moti Physio 做法 | 我方可借鉴 |
|--------|------------------|------------|
| **动作视频 + 3D + 椭圆** | 过顶深蹲：左侧动作视频（可切换理想姿势）、右侧 3D、椭圆（95% 置信区间）及角度 | 动态评估报告同时提供：原始动作回放、3D 动作重建、运动轨迹/置信区间等可视化 |
| **异常解释 + 解决方案** | 每个不平衡项可查看：异常释义、程度、影响因素、解决方案视频 | 每个异常项配套「释义 + 程度 + 影响因素 + 解决方案（含视频）」形成闭环 |
| **参考角度** | 各评估项（如膝不对称、上身前倾、过度后伸）提供 ⓘ 参考角度与理想值（如 93°） | 动态报告中的角度类指标均提供参考范围与理想值说明 |

## 三、综合借鉴清单（可直接落需求）

### 3D 模型（骨骼、肌肉）

- [ ] **骨骼 3D**：多视角、红区高亮失衡程度、与失衡列表联动定位
- [ ] **肌肉 3D**：红/蓝编码（紧张/无力）+ 深浅分级，可点击查失衡说明
- [ ] **双模式**：同一场景下骨骼模式 / 肌肉模式切换
- [ ] **三状态对比**：标准 / 当前 / 恶化 三种姿态对比，底部切换
- [ ] **理论标注**：肌肉状态与 Janda、Anatomy Trains 等理论可关联（文案或弹窗）

### 报告内容

- [ ] **计分屏**：总分 + Top 3 + 风险等级作为报告入口
- [ ] **多视图**：体态视图支持照片 / 轮廓(Moiré) / 面部分析 / 专项试验等切换
- [ ] **Top 3 联动**：点击 Top 3 项 → 图片/3D 对应位置高亮
- [ ] **Reference Guide**：关键指标 ⓘ 释义 + 参考范围
- [ ] **脊柱专项**：VCM 类力线视图 + Cobb 类角度视图，可切换
- [ ] **脊柱裸机模式配置**：设置中提供裸机模式开关，适用中重度脊柱侧弯场景，并配套测量前着装指引（如无上衣+紧身下装）
- [ ] **前后对比**：按改善程度排序、多维度筛选、3D 定位改善区域、Before/After 切换
- [ ] **多报告模板**：骨骼表、专家表、原始照片表、风险排名表等
- [ ] **分享**：人脸马赛克、机构 Logo/名称、「当前选中视图即分享内容」
- [ ] **动态报告**：动作视频 + 3D + 轨迹/椭圆 + 异常释义 + 影响因素 + 解决方案视频
