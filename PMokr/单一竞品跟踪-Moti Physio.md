## 2. 产品定位与核心卖点（必填）

**一句话定位**：韩国首款基于 RGB-D 相机的 3D 体态分析仪，面向医疗/康复级场景，提供与 X 光对标、无辐射的快速体态评估。

**官方主打卖点（Top3）**：
1. 医学级准确性：江南 Severance 医院 SCIE 论文验证，与 EOS X 光对比具有中等到良好的相关性
2. 30 秒非接触测量：无需标记点，无辐射，孕妇可用
3. 三阶段体态可视化 + 肌肉色彩编码（红=紧张/蓝=无力）+ 动态评估（过头深蹲、单腿站立）

**你理解的真实核心价值**：
- ☑ 测得准：有医学论文背书，专业认可度高

### 5.2 软件使用流程

整体流程分为**检测前准备 → 选择检测类型 → 执行测量 → 查看与分享报告**四个阶段。

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│  检测前准备      │ → │  选择检测类型    │ → │  执行测量       │ → │  报告查看与分享  │
│  着装/姿态确认   │    │  静态/核心/平衡  │    │  按屏幕指引完成  │    │  打印/邮件/二维码 │
└─────────────────┘    └─────────────────┘    └─────────────────┘    └─────────────────┘
```

**1. 检测前准备**：紧身衣物、脱鞋、长发扎成圆髻；保持放松；严重脊柱侧弯可开启 Bare Mode（Setting > Features > Bare Mode）。

**2. 选择检测类型**

| 检测类型 | 用途 | 测量方式 |
|----------|------|----------|
| 体态分析 | 发现体态失衡、强调矫正必要性 | 正面、侧面、背面 + Adam 前屈试验 |
| 核心功能分析 | 评估核心稳定性 | 5 次过头深蹲 |
| 平衡能力分析 | 评估平衡、预防跌倒 | 单腿站立 10 秒 |

**3. 执行测量（体态分析）**：正面/侧面/背面自然站立；Adam 前屈：膝盖伸直、下巴贴近锁骨、手臂前伸至指尖略高于髌骨；屏幕有实时姿态引导，约 30 秒完成。

![测量姿态示意](https://cdn.imweb.me/thumbnail/20251020/ec11af1b83576.gif)
*正面/侧面/背面及 Adam 前屈姿势 | 来源：[Measurement 页面](https://en.motiphysio.com/Measurement)*

**4. 报告查看与分享**：软件内查看 3D 结果、肌肉分析、运动建议；Moti Physio 2 支持邮件、二维码、社交报告。

### 5.3 软件核心功能（与 5.1 一/二/三级架构对应）

#### 一级：测量与建模

| 二级功能 | 核心能力说明 |
|----------|--------------|
| **静态测量** | 正面/侧面/背面三视角扫描，自然站立、目视前方；Adam 前屈试验（膝盖伸直、下巴贴近锁骨、手臂前伸至指尖略高于髌骨），用于脊柱侧弯筛查；屏幕实时姿态引导，单次约 30 秒完成 |
| **动态测量** | 5 次过头深蹲评估核心稳定性；单腿站立 10 秒（左/右分别测）评估平衡与跌倒风险；动作按屏幕指引完成，支持实时/回放查看 |
| **3D 建模** | RGB-D 深度数据驱动骨骼 3D 重建；肌肉 3D 重建与张力推断；支持正/侧/背多视角切换 |

#### 一级：体态分析（静态）

| 二级功能 | 核心能力说明 |
|----------|--------------|
| **计分屏** | 整体失衡分数 0–100（分数越高风险越大）；Top 3 失衡区域及角度值；风险等级 0–33 轻度 / 34–66 中度 / 67–100 重度；作为报告入口与咨询抓手 |

| **3D 模拟** | 失衡列表（左侧按严重度排序，点击跳转 3D 区域）；骨骼模式（红色高亮失衡程度）；肌肉模式（红=紧张、蓝=无力，颜色深浅=程度，点击弹出失衡说明）；标准/当前/恶化对比（强调矫正必要性） |

## 9. 附录：Janda 与 Anatomy Trains 理论说明

Moti Physio 的肌肉失衡分析与改善建议基于上述理论体系。以下为两套理论的原理及主要文献参考。

### 9.1 Janda 肌肉失衡理论

**理论创始**：捷克神经学家、物理治疗师 Vladimir Janda（弗拉迪米尔·扬达）博士于 1960 年代提出，基于肌电图研究区分肌肉功能类型。

**核心原理**

| 概念 | 说明 |
|------|------|
| **姿势肌（Tonic muscle）** | 过度活跃、抗疲劳能力强，在抵抗重力和保持直立中起重要作用；易紧张、缩短 |
| **相位肌（Phasic muscle）** | 易疲劳，仅在需要时启动；易无力、抑制 |
| **失衡机制** | 姿势肌与相位肌失衡导致可预测的姿势代偿与慢性疼痛 |

**三种肌肉失衡综合征**

| 综合征 | 典型表现 | 紧张肌肉 | 薄弱肌肉 |
|--------|----------|----------|----------|
| **上交叉症候群（UCS）** | 头部前倾、圆肩 | 上斜方肌、肩胛提肌、胸锁乳突肌、胸大肌 | 颈深屈肌、下斜方肌、前锯肌 |
| **下交叉症候群（LCS）** | 骨盆前倾、腰椎前凸增加 | 髂腰肌、竖脊肌 | 腹肌、臀肌 |
| **层状症候群** | 上、下交叉叠加 | — | — |

![Janda 上交叉症候群示意](https://www.physio-pedia.com/images/thumb/3/31/Jandas-Upper-Crossed-Syndrome-Permission-of-Human-Kinetics.png/400px-Jandas-Upper-Crossed-Syndrome-Permission-of-Human-Kinetics.png)
*Janda 上交叉症候群：紧张肌肉（实线）与薄弱肌肉（虚线）分布 | 来源：[Physiopedia](https://www.physio-pedia.com/Upper-Crossed_Syndrome)*

**评估与治疗要点**：强调功能性视角（运动模式、姿态、步态）而非单纯结构；治疗策略包括外周结构正常化、重建肌肉平衡、感觉运动训练；核心原则为「强化弱侧、拉长紧张侧」。

**主要文献**：
- Page P, Frank C, Lardner R. *Assessment and Treatment of Muscle Imbalance: The Janda Approach*. Human Kinetics, 2009. 该书为 Janda 理论实践应用的系统性文献。
- PMC 综述：*Assessment and Treatment of Muscle Imbalance – The Janda Approach* (Bodywork Movement Therapies, 2009)

### 9.2 Anatomy Trains（解剖列车）理论

**理论创始**：美国手法治疗师 Thomas W. Myers 于 2001 年首次出版，将人体肌筋膜系统比喻为铁路网络，用于理解力的传递与姿势代偿。

**核心原理**

| 概念 | 说明 |
|------|------|
| **车站** | 肌肉在骨上的附着点（近关节处） |
| **轨道** | 肌筋膜连结路径，串联一组肌肉 |
| **火车** | 动作产生的「力」，沿特定轨道在全身传递 |

**七大筋膜经线**：浅背线、浅前线、体侧线、螺旋线、手臂线、功能线、深前线。每条经线串联多块肌肉，形成力的传递路径；局部问题可能源于整条经线的张力失衡。

> 📖 **筋膜经线示意图**：可参考 [Anatomy Trains 官方海报](https://www.anatomytrains.com/product/anatomy-trains-3rd-ed-posters-large/) 或《解剖列车》教材中的经线图示。

**临床应用**：例如足底筋膜炎可能源于整条「浅背线」的肌筋膜张力失衡；治疗需考虑整条经线而非单一肌肉。

**主要文献**：
- Myers TW. *Anatomy Trains: Myofascial Meridians for Manual and Movement Therapists*. 首版 2001 年；第四版 2021 年（台版：爱思唯尔，中文版：北京科学技术出版社）。
- 第四版补充：筋膜组织与粘连图解、筋膜研究新证据、运动与拉伸内容。肌筋膜链理论在康复、软组织损伤、疼痛治疗领域应用广泛。
