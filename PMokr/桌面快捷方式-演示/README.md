# 桌面快捷方式 - 移动端演示

演示在手机/平板上「询问是否创建桌面快捷方式」的完整流程，包含**支持**与**不支持**两种场景及多浏览器操作指引。

## 体验链接

- **部署到已有仓库（子目录）**：  
  **https://你的用户名.github.io/仓库名/本演示所在子目录名/**  
  例如：`https://zhangsan.github.io/PMokr/desktop-shortcut-demo/`（不影响仓库内其他已部署内容）

- **部署到新仓库（根目录）**：  
  **https://你的用户名.github.io/仓库名/**  
  例如：`https://zhangsan.github.io/desktop-shortcut-demo/`

## 功能说明

- **① 支持创建桌面快捷键流程**：征询弹窗 → 同意 → 可调起系统「添加到主屏幕」或展示引导。
- **② 不支持创建桌面快捷键的流程**：征询弹窗 → 同意 → 直接展示当前浏览器添加步骤（不调起系统）→ 可查看其他浏览器操作方式。
