# 产品辅料设计规范

**版本**：V1.0  
**编制**：产品部  
**日期**：2025年2月  
**状态**：试行

## 一、文档说明与适用范围

### 1.1 目的

- 统一各产品线、各型号辅料应包含的**内容项**与**内容规范**。
- 明确每类辅料的**必选/可选**属性，以及**豁免场景**，便于研发、生产、质量与市场协同执行。
- 为后续辅料模版迭代、新品上市辅料配置提供依据。

### 1.2 适用范围

- **产品线**：医疗产品线、商业产品线（定义见第二节）。
- **辅料类型**：铭牌、合格证、使用说明海报、保修卡等随产品交付的印刷品或电子替代物。
- **适用角色**：产品经理、工业设计、质量、生产、法务、市场。

### 1.3 与现有模版的对应关系

本规范基于《辅料样式内容汇总》中现有模版整理，规范中“必选/可选/豁免”的界定在试行期内可与现有配置并存，逐步统一。

## 二、产品线定义

| 产品线 | 定义 | 典型系列/型号示例 | 合规与资质要求 |
|--------|------|-------------------|----------------|
| **医疗** | 用于医疗机构、体检中心等，涉及人体成分、体态、医学图像处理等医疗/健康评估场景的产品 | VE73X（VE725/730/735/732）、VE760 及延展型号（VE760/755/761/768）等含“体态医学图像处理”“人体成分分析”等描述的产品 | 需满足医疗器械或相关法规对标识、说明书、合格证等要求 |
| **商业** | 用于健身房、康体中心、企业 wellness 等非医疗场景的产品 | VE710、VE310、VE560、VE560 Pro 等 | 需满足消费品标识、三包及企业品牌规范 |

**说明**：同一物理硬件若同时面向医疗与商业渠道，应按**较高要求**（医疗线）配置辅料，或按渠道分别定义辅料清单。

## 三、辅料类型总览

| 辅料类型 | 说明 | 常见载体 |
|----------|------|----------|
| 铭牌 | 产品/部件名称、型号、关键参数、制造商等信息的固定标识 | 机身贴纸、金属/塑料铭牌、包装印刷 |
| 合格证 | 出厂检验合格及符合性声明 | 纸质卡片、贴纸、或包装/说明书上的合格声明区 |
| 使用说明海报 | 安装、使用、注意事项的图示化说明，常张贴于使用现场 | 单页/多页海报、展架卡纸、电子版 PDF |
| 保修卡 | 保修条款、期限、保修凭证信息 | 独立卡片、说明书内页、或电子凭证 |

## 四、各辅料类型设计规范

### 4.1 铭牌

#### 4.1.1 应包含的内容

| 内容项 | 内容规范 | 是否必须 | 豁免场景 |
|--------|----------|----------|----------|
| 产品/部件名称 | 与注册或上市名称一致；可区分“整机名称”与“部件名称”（如软件、硬件、人体成分分析仪、体态医学图像处理软件等） | **必选** | 无 |
| 型号/规格 | 与产品型号一致（如 VE725、VE730、VE760 等），多部件时可为“主机型号 + 部件型号” | **必选** | 无 |
| 制造商名称 | 公司全称 | **必选** | 无 |
| 制造商地址 | 注册地址或主要经营地址 | **必选** | 无 |
| 生产日期或序列号 | 可仅标生产日期，或“生产日期 + 序列号/出厂编号” | **必选** | 纯软件且无独立包装时，可在软件界面或授权书中体现，不强制要求物理铭牌 |
| 电源/电气参数 | 额定电压、频率、功率等（若适用） | **必选**（硬件） | 纯软件、无外露电源的部件可不标 |
| 产品标准/注册证编号 | 医疗器械注册证号、产品技术要求编号或执行标准（医疗线） | **医疗线必选**；商业线建议 | 商业线若未取得医疗器械注册，可不标注册证号，但建议标执行标准 |
| 软件版本或环境说明 | 如“体态医学图像处理软件（硬件环境）”等，便于与主机区分 | **可选** | 无独立铭牌位置的软件可仅在说明书/合格证中体现 |

#### 4.1.2 设计规范（通用）

- 文字清晰可读，字体、字号符合可读性及行业惯例。
- 铭牌信息与说明书、合格证、包装上的产品名称、型号、制造商保持一致。
- 医疗线铭牌内容需与注册/备案信息一致，不得擅自变更表述。

#### 4.1.3 与现有模版的对应

- 现有模版中“铭牌”列填写内容即产品/部件名称（如“3D体型体态及人体成分评估系统”“软件”“硬件”“人体成分分析仪”“体态医学图像处理软件（硬件环境）”），规范要求在上述名称基础上，铭牌上还须包含：型号、制造商、生产日期/序列号、电气参数（若适用）、医疗线注册/标准信息等。

### 4.2 合格证

#### 4.2.1 应包含的内容

| 内容项 | 内容规范 | 是否必须 | 豁免场景 |
|--------|----------|----------|----------|
| 产品名称 | 与铭牌、说明书一致 | **必选** | 无 |
| 型号/规格 | 与铭牌一致 | **必选** | 无 |
| 出厂编号/序列号 | 可追溯至生产批次或单机 | **必选** | 纯软件且无单机序列号时，可用“授权版本/批次”替代 |
| 检验结论 | 明确“合格”或“经检验合格”等表述 | **必选** | 无 |
| 检验日期 | 与生产/出厂日期逻辑一致 | **必选** | 无 |
| 检验员签章或代号 | 可追溯 | **建议** | 若采用电子合格证或系统打印，可用唯一编号替代签章 |
| 制造商名称 | 与铭牌一致 | **必选** | 无 |
| 符合性声明 | 如“符合产品技术要求/GB/行业标准” | **医疗线必选**；商业线建议 | 商业线可简化为“符合产品标准” |

#### 4.2.2 设计规范（通用）

- 合格证须随产品或包装一并交付，确保用户/客户可获取。
- 电子合格证若替代纸质，须可下载、可留存，且包含上述必选项。

#### 4.2.3 与现有模版的对应

- 现有《辅料样式内容汇总》中“合格证”列多为空，表示当前多为统一配置。规范生效后，建议所有出厂产品均配备合格证（纸质或电子），内容按上表执行。

### 4.3 使用说明海报

#### 4.3.1 应包含的内容

| 内容项 | 内容规范 | 是否必须 | 豁免场景 |
|--------|----------|----------|----------|
| 产品名称与型号 | 与铭牌一致 | **必选** | 无 |
| 安全注意事项/警示 | 用电、环境、使用人群、禁忌症等（按产品风险） | **必选** | 风险极低、无特殊禁忌的商业产品可做简化，但不得省略 |
| 安装/摆放说明 | 图示化安装步骤、环境要求 | **必选**（硬件） | 纯软件、无安装步骤时可省略安装说明，保留“运行环境”说明 |
| 使用步骤/操作指引 | 简明操作流程，图示为主 | **必选** | 无 |
| 日常维护与保养 | 清洁、存放、简单故障排查 | **建议** | 若说明书已含且用户易获取，海报可仅做要点提示或省略 |
| 制造商/售后服务联系方式 | 电话、网址、二维码等 | **必选** | 无 |
| 版本/日期 | 便于与产品版本对应 | **建议** | 无 |

#### 4.3.2 设计规范（通用）

- 以图示为主，文字精简，适合现场张贴、快速查阅。
- 医疗线须包含与说明书一致的禁忌症、警示语，不得与注册说明书冲突。
- 可与《使用说明书》内容互补：海报侧重现场快速指引，说明书侧重完整法律与技术信息。

#### 4.3.3 与现有模版的对应

- 现有模版中“使用说明海报”列部分型号为“/”，表示该型号当前未配置或可不配置。规范建议：**硬件产品原则上均配置使用说明海报**；纯软件且无独立硬件交付时，可在“豁免”下不单独提供海报，但须在软件内或电子说明中提供等效的安装/使用指引。

### 4.4 保修卡

#### 4.4.1 应包含的内容

| 内容项 | 内容规范 | 是否必须 | 豁免场景 |
|--------|----------|----------|----------|
| 产品名称与型号 | 与铭牌一致 | **必选** | 无 |
| 出厂编号/序列号（或保修凭证号） | 与合格证一致，用于保修绑定 | **必选** | 无序列号产品可用“购机日期+渠道”等作为凭证 |
| 保修期限 | 起算方式（如出厂日/购机日）、保修时长 | **必选** | 无 |
| 保修范围与除外条款 | 哪些部件/情形在保、哪些不在保 | **必选** | 可引用完整条款的链接或二维码 |
| 保修流程/联系方式 | 报修方式、客服电话、网点或二维码 | **必选** | 无 |
| 制造商/销售商信息 | 与铭牌、合同一致 | **必选** | 无 |
| 用户填写区 | 购机日期、用户名称、联系方式等（可选） | **可选** | 若采用电子建档，可不设手写区 |

#### 4.4.2 设计规范（通用）

- 保修政策须与销售合同、官网公示一致；医疗线若有法规要求的保修/售后服务，须在保修卡或说明书中体现。
- 可独立成卡，也可合并于说明书内页或电子凭证，但内容须完整可查。

#### 4.4.3 与现有模版的对应

- 现有模版中“保修卡”列多为空，表示统一配置或未在表中展开。规范要求：**所有面向最终用户销售的产品均应提供保修承诺及凭证**（卡片或电子），内容按上表执行。B2B 批量采购若合同另有约定，可按合同执行，但建议仍提供最小化保修说明。

## 六、实施与迭代

### 6.1 实施建议

1. **新品**：自本规范发布后立项的产品，辅料设计按本规范执行，并在需求/评审中明确“辅料清单与必选项”。
2. **在售产品**：在下次辅料改版或包装/说明书换版时，逐步对齐本规范；医疗线优先对齐铭牌、合格证、使用说明中的合规项。
3. **辅料模版维护**：建议在《辅料样式内容汇总》或等效表中增加“是否必选”“豁免场景”列，并与本规范版本号对应。

### 6.2 豁免申请

若某型号因技术、成本或渠道原因需豁免某类辅料或某必选内容项，须由产品经理提出申请，经质量/法务会签后归档，并在本规范附录或辅料表中注明例外及理由。

### 6.3 规范修订

- 规范修订由产品部牵头，涉及医疗合规的变更须经质量/注册确认。
- 修订后更新版本号与修订说明，并同步至《辅料样式内容汇总》及设计/生产相关流程。
