# WellnessHub產品使用說明書
wellnesshub專為運動健康機構打造，集設備管理、體質評估、智能分析、训練規劃于一體。通過自動化連接體測設備，實時生成多維健康報告并預警異常指標，AI智能生成个性化训練方案，同時為教練提供会員資料追踪与趨勢分析工具，助力機構實现“評估-異常分析-个性化訓練大綱-優化”的全流程闭环管理。讓科學資料驅動精准服務，用智能工具提升運營效能，助您打造更具競爭力的健康管理服務體係。

### 報告列表
展示綁定的所有設備測量后的報告資料，可查看測量報告、列印報告、刪除資料。支持按照日期用戶手機號、信箱、姓名进行查詢。列表支持按照門市名称、設備类型、設備序列表篩選。

#### 列表資料
+ 掃描時间：該報告用戶掃描的時间。
+ 所屬門市：当前報告的所屬門市資訊。
+ 設備序號：顯示測量設備的序號。
+ 姓名：測量用戶姓名。
+ 信箱/手機號：測量用戶注册時使用的手機號/信箱。
+ 性別：測量用戶性別。
+ 工作人員：当前用戶已分配给的工作人員帳號。若没有分配工作人員，則不顯示。
+ 操作欄：
    - 體測報告：点擊后可查看體測報告詳情。
    - 刪除：可刪除報告資料，刪除后資料不可恢複。
    - 下載模型：支持下載體態、體圍模型。<font style="color:#DF2A3F;">僅对開通該權限的客戶開放。</font>
    - AI報告解读：若為Basic版本，則AI報告解读按钮置灰不可点擊。

<font style="color:#DF2A3F;">注：若帳號權限為【</font><font style="color:#DF2A3F;background-color:rgb(242, 243, 245);">分配權限-工作人員</font><font style="color:#DF2A3F;">】權限，則可查看所有用戶報告資料。</font>

在用戶列表中点擊某用戶的【健康檔案】，可进入該用戶的健康檔案頁。健康檔案用于记錄用戶的基础資訊、健康背景、生活方式等，填寫后生成訓練大綱時会結合健康檔案与最近一次測量報告，生成更贴合該用戶的训練方案。頁面支持【編輯】与【生成訓練大綱】。

**健康檔案字段說明：**

1. 基础資訊：維護用戶基础資訊。

+ 包含字段： 头像、昵称、性別、年龄、身高、體重、信箱、手機號。
+ 信箱/手機號的特殊說明： 作為用戶注册時的唯一標识，只能在用戶端进行修改。

2. 健康背景：收集用戶的健康資訊。

+ 疾病史： 记錄用戶的疾病史。
+ 目前用藥： 此字段為「選填」項，要求用戶输入目前用藥情况，最多500字。
+ 運動禁忌： 此字段也為「選填」項，提供示例說明（如：避免深蹲、禁止高强度有氧），最多500字。

3. 生活方式：反映用戶日常活動与睡眠情况。

+ 日常活動量： 下拉選擇。
+ 睡眠時長： 下拉選擇。

4. 飲食習慣：记錄用戶飲食相关偏好与限製。

+ 每日餐数： 此字段為「選填」項，僅可输入数字0–20，單位：餐。
+ 飲食要求或偏好： 下拉選擇。
+ 偏好食材： 此字段為「選填」項，可填寫示例（如：偏爱牛肉/討厭蔬菜），最多500字。

5. 運動目標与偏好：记錄用戶訓練目標与身體注意事項。

+ 訓練目標： 下拉選擇。
+ 訓練頻率： 下拉選擇。
+ 單次训練時长： 下拉選擇。
+ 重点训練部位： 下拉選擇。
+ 需注意的傷病部位： 下拉選擇。
+ 形體管理目標： 下拉選擇（根據門市 AI 解读維度配置顯示）。
+ 運動能力提升需求： 下拉選擇（根據門市 AI 解读維度配置顯示）。

<font style="color:#df2a3f;">注：形體管理目標、運動能力提升需只在門市資訊管理AI解读維度選擇【形體美學/運動能力】的時候顯示；</font>

6. 訓練大綱偏好：用于生成訓練大綱時的周期与階段設置。

+ 训練周期： 下拉選擇，默認：1个月（4周）。
+ 階段数量： 下拉選擇，默認：3階段。

7. 補充資訊：其他需補充的内容。

+ 補充資訊： 此字段為「選填」項，可填寫训練經驗、基础疾病、特殊训練需求或其他相关資訊，最多500字。
